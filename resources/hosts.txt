# ft_nmap hosts file
# Format: <ip> <ports> [scan_types]
#
# ip: IP address or hostname
# ports: range (1-1024) or comma-separated (80,443,8080)
# scan_types (optional): SYN,FIN,NULL,XMAS,ACK,UDP (if omitted, all types are used)

# Example hosts:
22.22.22.22 1-10 ACK,UDP
scanme.nmap.org 22,80,99-100 SYN,UDP
yandex.ru -100 SYN,ACK
google.com 1,2,3
